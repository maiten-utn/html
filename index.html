1. Estructura básica del documento HTML

La estructura de HTML se basa en el uso de etiquetas que definen el contenido y su organización en una página web.
Estas etiquetas, encerradas en corchetes angulares, vienen en pares para marcar el inicio y el fin de cada elemento.
HTML utiliza una jerarquía de elementos, que incluye contenedores como div y header, así como elementos en línea
como span y a para estructurar y formatear el contenido de manera organizada. Esta estructura permite a los navegadores
interpretar y mostrar correctamente las páginas web.
